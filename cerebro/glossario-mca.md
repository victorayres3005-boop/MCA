# Glossário — Plataforma MCA

## Entidades do domínio

| Termo | Definição |
|---|---|
| **MCA** | Consultora — usuária principal do SaaS. Atua como gestora intermediária entre clientes e contratadas. |
| **Cliente** | Empresa contratante. Define estratégia, aprova orçamento, recebe relatórios consolidados. |
| **Contratada** | Construtoras, projetistas, fornecedores de serviços/equipamentos, consultorias. |
| **Carteira** | Conjunto de projetos gerenciados pela MCA simultaneamente. |
| **Gerente de projeto** | Usuário da MCA que gerencia a obra. Role `gerente` no sistema. |

## Termos PMBOK

| Termo | Definição |
|---|---|
| **PMBOK** | Project Management Body of Knowledge. Metodologia de referência para gerenciamento de projetos. |
| **TAP** | Termo de Abertura do Projeto. Documento que autoriza formalmente o projeto. |
| **TEP** | Termo de Encerramento do Projeto. Documento de aceite formal das entregas. |
| **PGP** | Plano de Gerenciamento do Projeto. Documento mestre com todos os planos subsidiários. |
| **RMA** | Relatório de Monitoramento e Acompanhamento. Status periódico do projeto. |
| **RACI** | Responsible, Accountable, Consulted, Informed. Matriz de responsabilidades. |
| **EAP** | Estrutura Analítica do Projeto (WBS). Decomposição hierárquica das entregas. |
| **MAS** | Matriz de Aquisições e Serviços. |
| **RNC** | Registro de Não Conformidade. Documento de qualidade. |
| **CapEx** | Capital Expenditure. Investimento de capital no projeto. |
| **Curva S** | Representação gráfica do progresso acumulado (prazo ou custo) ao longo do tempo. |
| **Semáforo** | Indicador visual verde/amarelo/vermelho de saúde de KPIs. |
| **Marco** | Milestone. Evento significativo ou ponto de controle no projeto. |

## Fases do projeto (PMBOK)

| Fase | Código | Descrição |
|---|---|---|
| Iniciação | `iniciacao` | Autorização e definição inicial |
| Planejamento | `planejamento` | Elaboração dos planos de gerenciamento |
| Execução | `execucao` | Implementação dos planos |
| Monitoramento e Controle | `monitoramento` | Acompanhamento, medição, correção |
| Encerramento | `encerramento` | Formalização da conclusão |

## Status de projeto

| Status | Significado |
|---|---|
| `planejamento` | Projeto em fase de planejamento |
| `execucao` | Em execução ativa |
| `monitoramento` | Fase de monitoramento e controle |
| `encerrado` | Concluído formalmente |
| `suspenso` | Paralisado temporariamente |

## Semáforos (padrão do sistema)

| Cor | Hex | Significado |
|---|---|---|
| Verde | `#16A34A` | No prazo / dentro do orçamento / conforme |
| Amarelo | `#F59E0B` | Em risco / atenção necessária |
| Vermelho | `#DC2626` | Atrasado / crítico / não conforme |
| Roxo | `#9333EA` | Encerrado |
| Cinza | `#6B7280` | Suspenso / inativo |
